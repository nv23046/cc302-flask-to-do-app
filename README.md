# To-Do App (Flask + Docker)

## Features
- Add, edit, and delete tasks
- SQLite-backed persistence
- Docker and Docker Compose support
- Basic CRUD tests with `pytest`

## Requirements
- Docker
- Docker Compose
- Git

## Run Locally
```bash
docker-compose up --build
```

Open: `http://localhost:5000`
