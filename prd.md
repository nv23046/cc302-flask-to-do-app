## Product Overview
A web-based To-Do application that allows users to manage daily tasks. Built with Python and Flask, containerized with Docker, and published to Docker Hub.

## Goals
- Create, view, edit, and delete tasks.
- Ensure consistent deployment using Docker.
- Practice full software development workflow.

## Target Users
Students learning web development and DevOps.

## Functional Requirements
- Add, view, edit, delete tasks.
- Store tasks using SQLite.
- Clean and simple UI.

## Non-Functional Requirements
- Use Flask.
- Use Docker and Docker Compose.
- Use GitHub for version control.
- Deploy image to Docker Hub.

## Tech Stack
- Python + Flask
- SQLite
- HTML/CSS
- Docker, Docker Compose
- GitHub, Docker Hub

## Success Criteria
- App runs locally and in Docker.
- CRUD works.
- Docker image is published successfully.
